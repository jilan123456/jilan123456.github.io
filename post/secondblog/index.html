学习笔记#

1.文件系统命令1#

pwd :显示当前工作目录的绝对路径#

cd :更改工作目录路径#

  • . 代表当前所在目录
  • .. 代表当前目录位置的上一层目录
  • ~ 代表家目录

mkdir :创建目录#

mkdir -p xxx //-p:若路径中的目录不存在，先创建目录

ls :列出目录和文件信息(-lh常用)#

ls -lh xxx //-h与-l一起，以易于阅读的格式输出文件大小

rmdir :删除空目录#

touch :创建空文件、更改文件时间#

cp :复制文件或目录#

cp jilan.txt ./

mv :文件和目录改名、移动文件和目录#

mv jilan.txt jil.txt

rm :删除文件或者目录。一般使用rm -rf删除目录(-rf常用)#

rm -rf jilan.txt //-f:强制删除、-r:递归删除目录

通配符#

ls ?.txt //?:匹配任意一个字符
ls *.txt //*:匹配任意多个字符(包含0个)
ls [abc]* //[]:相当于或的意思 
ls [a-c]* //-:代表一个范围，比如a-z

2.Vim编辑器#

安装vim#

sudo yum install vim

Vim命令模式基本操作#

  • i 插入
  • Esc 退出插入
  • :w 保存
  • :q 退出
  • :q! 强制退出
  • /word 从上而下查找字符串"word"
  • n 定位下一个匹配被查找字符串
  • N 定位上一个匹配被查找字符串
  • u 取消最近一次操作
  • 删除: x dd #dd
  • 替换: r ``
  • 复制: yy #yy
  • 粘贴: p P
  • 行号显示: :set nu :set nonu

Vim底行模式基本操作#

  • 保存文件: :w
  • 退出vi: :q :q!
  • 保存文件退出vi: :wq

3.文件系统命令2#

find :查找文件或目录#

find /usr/bin -name "host*" -type f //f普通文件，d目录
find /usr/bin -size +1M

which :用来搜索二进制文件、可执行文件或者shell命令行的位置#

which find
which cp

cat :显示文件内容、连接文件内容#

cat a.txt b.txt

head :显示指定文件的前若干行#

head -n 3 /etc/passwd

tail :查看文件末尾数据#

tail -n 3 /etc/passwd

grep :搜索与字符串匹配的行#

grep user /etc/passwd
  • -i 查找时忽略大小写 grep -i user /etc/passwd
  • -n 显示行号 grep -n user /etc/passwd
  • -r 递归搜索子目录 grep -r user /etc/passwd
  • -v 反转查找 grep -v user /etc/passwd

tar :对文件进行打包、解包#

tar -cjvf jilan.tar.xz jilan
tar -xjvf jilan.tar.xz

zip :打包目录、解压目录#

zip -r jilan.zip dir //-r表示递归打包目录
unzip jilan.zip

4.重定向和管道#

输入重定向#

python a.py < b.txt

输出、错误重定向#

  • 重定向标准输出 :> >>
  • 重定向标准错误 :2> 2>>
  • 重定向标准输出和标准错误 :&> &>>

echo命令#

echo "nihaoya"
echo "nihaoya" >b.txt
echo "nihaoya" 2>>b.txt

管道grep :管道可以把一系列命令连接起来#

ls -l | grep jilan

wc :可以显示行数、单词数和字符数#

wc -l jilan.py //仅显示多少行
wc -w jilan.py //仅显示多少字
wc -m jilan.py //多少字符

xargs :命令传递参数的一个过滤器#

ls /usr/bin/a* | xargs
find ./ -name *.conf | xargs wc -l

5.用户和权限管理#

root用户 :拥有Linux系统的最高权限#

/etc/passwd :用于保存用户的帐号基本信息#

su :可以在任意用户之间进行切换#

添加用户 :#

useradd linux
grep linux /etc/passwd
ls /home
passwd linux

删除用户#

userdel -r linux
ls /home/

添加组#

groupadd web
grep web /etc/group

将用户添加到一个组#

usermod -a -G web linux
grep web /etc/group

删除组#

groupdel web
grep web /etc/group

6.进程管理和其他命令#

kill :用于终止一个进程#

kill xx

crontab :编辑当前用户的计划任务#

crontab -e

free :显示内存的使用情况#

free -h

df :查看磁盘使用情况#

df -h

du :显示指定的目录或文件所占用的磁盘空间#

du -sh linux-5.9.1.tar.xz //-s 显示总计结果

安装gcc#

sudo yum install gcc

使用Gdb调试#

gcc -o main -g main.c
./main

8.Shell脚本编程#

Shell脚本#

bash hello.sh

$ 变量符#

echo ${USER}.jpg

数组#

array_name=(value1 value2 ... valuen)

9.sed#

's/ /-/g' jilan.txt //替换1
's/this/----/i' jilan.txt //替换2忽略大小写
's/this/----/p' jilan.txt //替换3-打印
sed '2,$s/ /-/g' jilan.txt //范围
sed -n '/^jilan p' jilan.txt //p 对匹配内容进行打印
sed '/^jilan/ d' jilan.txt //d 删除匹配内容
sed '/^jilan/ a xxx' jilan.txt //a 新增一行
sed '/^jilan/ i xxx' jilan.txt //i 插入一行

10.awk#

awk支持从管道读取内容#

echo "1 2 3" | awk '{print $2}'
  • BEGIN 用于匹配第一个输入文件的第一行之前的位置
  • 内置变量NR表示当前行号
  • END 则用于匹配处理过的最后一个文件的最后一行之后的位置
